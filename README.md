# Pete McPherson Portfolio

A modern personal brand portfolio for Pete McPherson, built as a conversion-focused landing experience with newsletter signup as a core CTA.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Pages

- Home: hero, positioning copy, featured projects, newsletter CTA
- About: story, services, philosophy, newsletter CTA
- Projects: project grid with outcomes and links, newsletter CTA

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` (or the port shown in terminal).

## Notes

- Newsletter form is front-end only (no backend integration yet).
- The site is optimized for clear positioning and conversion-focused UX.
